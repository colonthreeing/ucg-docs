Uncanny Cat Golf is made in the Godot Engine version 4.4.1. As such, you will need to download it to mod the game. You can download it [here](https://godotengine.org/download/archive/4.4.1-stable/). Most likely you will want the Standard version, the `.NET` version doesn't include anything useful for modding UCG.

Once you have downloaded Godot, you can open it up. We'll need it after we finish decompiling the game. In the future, the game's source code will be released publicly on [this github page](https://github.com/SlappyHappy2000/UCG-Release-Builds), so you won't have to decompile it and can simply import the project.
